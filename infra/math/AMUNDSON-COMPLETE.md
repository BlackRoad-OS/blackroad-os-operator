# PART 6: ALGEBRAIC IDENTITIES REFERENCE

## Square Identities

```
(a+b)² = a² + 2ab + b²
(a-b)² = a² - 2ab + b²
a² - b² = (a+b)(a-b)
```

## Cube Identities

```
(a+b)³ = a³ + 3a²b + 3ab² + b³
(a-b)³ = a³ - 3a²b + 3ab² - b³
a³ + b³ = (a+b)(a² - ab + b²)
a³ - b³ = (a-b)(a² + ab + b²)
```

## Triple Sum

```
(a+b+c)² = a² + b² + c² + 2ab + 2bc + 2ca
a³ + b³ + c³ - 3abc = (a+b+c)(a² + b² + c² - ab - bc - ca)
```

**Special case:** If a + b + c = 0, then a³ + b³ + c³ = 3abc

## Exponent Rules

```
aᵐ × aⁿ = aᵐ⁺ⁿ
aᵐ / aⁿ = aᵐ⁻ⁿ
(aᵐ)ⁿ = aᵐⁿ
(ab)ᵐ = aᵐbᵐ
a⁻ᵐ = 1/aᵐ
```
