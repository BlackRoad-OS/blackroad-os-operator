## The Complete Lagrangian

```
Z = ∫ DA Dψ Dψ̄ Dφ exp(iS)
```

Where the action S contains:

### Gauge Field Terms

```
-¼ Fᵃμν Fᵃμν
```

Field strength tensor for gauge bosons (gluons, W±, Z, photon)

### Fermion Kinetic Terms

```
ψ̄ᵢ iγᵘ Dμ ψᵢ
```

Dirac equation for quarks and leptons with covariant derivative

### Yukawa Coupling

```
-yᵢⱼ ψ̄ᵢ φ ψⱼ + h.c.
```

Fermion-Higgs coupling generating masses

### Higgs Potential

```
|Dμφ|² - V(φ)

V(φ) = μ²|φ|² + λ|φ|⁴
```

Mexican hat potential for spontaneous symmetry breaking

## Components

| Symbol | Meaning |
|--------|---------|
| Z | Partition function |
| DA | Gauge field integration measure |
| Dψ, Dψ̄ | Fermion field measures (Grassmann) |
| Dφ | Scalar (Higgs) field measure |
| Fᵃμν | Field strength tensor |
| γᵘ | Dirac gamma matrices |
| Dμ | Covariant derivative |
| yᵢⱼ | Yukawa coupling matrix |
| φ | Higgs doublet |

## Symmetry Group

```
SU(3)_C × SU(2)_L × U(1)_Y
```

- **SU(3)_C**: Color (strong force)
- **SU(2)_L**: Weak isospin
- **U(1)_Y**: Hypercharge

After electroweak symmetry breaking:
```
SU(2)_L × U(1)_Y → U(1)_EM
```

## Connection to Amundson Framework

The Standard Model Lagrangian embodies all four primitives:

| Primitive | SM Manifestation |
|-----------|------------------|
| Change (Ĉ) | Time evolution, interactions |
| Strength (Ŝ) | Coupling constants (g, g', yᵢⱼ) |
| Structure (Û) | Gauge symmetry groups |
| Scale (L̂) | Mass generation via Higgs VEV |

The path integral Z is the ultimate Z = ∅ equation — summing over all possible field configurations to find equilibrium.
