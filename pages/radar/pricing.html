Questions

What counts as an event?
Each HTTP call logged to the telemetry API counts as one event. This includes calls from your agents, services, and applications using the TracedClient.
What happens if I exceed my event limit?
On paid plans, you'll be charged the overage rate per 1,000 additional events. On the free plan, logging stops until the next billing period.
What is the "New Internet"?
The New Internet Radar identifies hosts outside the traditional big tech ecosystem (Google, Meta, Amazon, etc.). These are the independent sites, APIs, and services building the decentralized future.
How do I integrate?
Drop in our TracedClient library (Python, TypeScript, or plain HTTP) and every outbound request is automatically logged. Takes about 5 minutes.
